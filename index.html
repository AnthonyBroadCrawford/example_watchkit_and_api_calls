Example Watchkit App w/ background API calls

A simple project illustrating how to have your watchkit extension (both Glance and App) interface with your iOS app to make a sucessful http api call to retrieve remote data

Why I created this example

I recently had to build a WatchKit app that connected to the parent app to retrieve data. What made this a bit tricky, is that the parent app would need to call a remote http api to obtain the necessary data.

Oddly enough, the necessary steps are not well documented and a lot of the information I found online was incomplete.

What this example includes

This example covers the following use-cases

  • Glance interface requesting remote data from iOS parent
  • Watch app requesting remote data from iOS parent
  • AppDelegate handleWatchKitExtensionRequest properly orchestrating a remote http api call

I've additionally thrown in some XCTTests' for the code in the project.
